## Этап 1 — Создание приложения одним промтом

**Промт из урока:**
```
Прочитай файлы в проекте:
research.md, brief.md.
На их основе создай Telegram Mini App.

Технические требования:
- HTML + CSS + JavaScript, без фреймворков
- Подключи Telegram Web App SDK (telegram-web-app.js) —
  для данных пользователя и темы Telegram
- Все файлы положи в папку tg-app/
- Точка входа — tg-app/index.html

Дизайн:
- Стиль нативного Telegram-приложения:
  закруглённые углы, мягкие тени,
  светлый фон, акцент (#2AABEE)
- Поддержка тёмной темы
- Выглядит как ПРИЛОЖЕНИЕ, не как сайт в рамке
- Плавные переходы между экранами (slide или fade, 200–300ms)

Мобильная адаптация:
- Только мобильный экран (320–430px)
- Кнопки минимум 44px в высоту
- Шрифт не менее 14px
- Нет горизонтальной прокрутки
- Касания отзываются визуально

Контент:
- Русский язык интерфейса
- Реальные данные из brief.md (не заглушки "Lorem ipsum")
- Если данных нет — придумай реалистичные для этой ниши

Дополнительно:
- Создай CLAUDE.md — какие файлы за что отвечают,
  навигация между экранами, где менять данные
- Комментарии на русском в каждом файле
- Задавай мне вопросы по ходу работы
```

**Задачи:**
- [ ] Отправить промт в Claude Code
- [ ] Дождаться создания папки `tg-app/` со всеми файлами
- [ ] Проверить — открыть `tg-app/index.html` двойным кликом
- [ ] Запустить локальный сервер: «Запусти приложение, хочу проверить»
- [ ] Проверить на телефоне через локальную ссылку (та же Wi-Fi сеть)
